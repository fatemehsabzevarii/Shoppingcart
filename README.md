## About The Project

For the shopping project, we  create  products. 

This project will do:
* register a customer
* login a customer
* Delete a product
* Create a product
* Get a product
* * Get all products
* Update a product
* Delete a product

In this project, we used the following technologies:
* [Python](https://www.python.org)
* [Mongo DB](https://www.mongodb.com)
